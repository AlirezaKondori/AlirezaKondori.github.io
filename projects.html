Projects

AI Note Taker

AI Note Taker is an AI-powered PDF note-taking application that enables users to upload PDF documents and generate coherent notes using the Groq API. The application features real-time progress tracking, user authentication, and a visually appealing UI. Key features include an efficient chunking strategy for handling large PDFs, real-time progress updates via server-sent events, secure login through Auth0, and the ability to download generated notes in PDF format. The modular codebase ensures optimized API usage and robust error handling, enhancing the overall user experience.

Password manager

The Password Manager project is a secure and efficient application I developed to manage sensitive credentials locally. It features encryption powered by Python's cryptography library, ensuring robust protection of stored passwords through a master key mechanism. The project includes essential functionality such as adding, retrieving, and listing passwords, all persisted in an SQLite database. Designed with security and usability in mind, this project demonstrates my proficiency in Python, cryptographic techniques, and database management while showcasing my commitment to building practical, user-focused software solutions.

ResumeAi

ResumeAI is an AI-driven application designed to enhance your resume by aligning it with specific job descriptions, thereby increasing your chances of securing interviews. Users can upload their existing resume along with a job description or a link to it, and the application intelligently adjusts various components of the resume to better match the job's requirements. By optionally providing additional details about past experiences, projects, and skills, users enable the application to more effectively tailor the resume to highlight relevant qualifications. This personalized approach ensures that your resume stands out by emphasizing the competencies most pertinent to the desired position.
